# 💸 IIITL Bachat

An AI-powered personal finance platform that helps users **track expenses, analyze spending, import receipts automatically, and explore investment insights** from one dashboard.

Built with the **MERN Stack + Gemini AI** to combine finance management with intelligent automation.

---

## ✨ Features

📊 Expense & Income Tracking  
🤖 AI Receipt / PDF Import using Gemini  
📈 Analytics Dashboard & Monthly Trends  
💬 AI Finance Assistant (Budgeting + Insights)  
📂 Excel Export for Transaction Records  
📉 Investment Planning & Educational Suggestions  
🔐 Google Authentication + Secure Login  
📱 Responsive Dashboard Experience  

---

## 📂 Project Setup

```bash
# Clone
git clone <repo-url>

# Backend
cd backend
npm install
npm run dev

# Frontend
cd frontend
npm install
npm start
```

Create `.env`

```env
MONGO_URL=
GEMINI_API_KEY=
GOOGLE_CLIENT_ID=
REACT_APP_API_URL=
```
